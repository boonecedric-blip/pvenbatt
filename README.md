# ⚡ PV & Batterij Calculator

Bereken uw besparing met zonnepanelen en thuisbatterij op dynamisch tarief (Belpex).

## Features

- 📊 Upload Fluvius CSV kwartierwaarden
- 📅 **Selecteer referentiejaar** uit geüploade data
- ☀️ PV simulatie met Belgisch zonnemodel
- 🔋 Slimme batterij optimalisatie
- 💰 Vergelijk 3 scenario's

## Installatie

```bash
npm install
npm run dev
```

## Build & Deploy

```bash
npm run build
```

Deploy naar Vercel: push naar GitHub en importeer in Vercel.
